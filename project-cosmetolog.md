# Система возврата клиентов для косметологов — Описание продукта

## Позиционирование
**Специалист по созданию системы возврата клиентов для косметологов под ключ**

Создаю сайты, квизы, чат-боты и автоматические напоминания — всё без участия клиента.

## Что это

Готовая система, которая возвращает потерянных клиентов косметолога автоматически.
Косметолог продолжает работать руками — система работает за него: напоминает, записывает, удерживает.

## Для кого

* Частные косметологи
* Косметологические кабинеты
* Небольшие бьюти-студии

В будущем:
* Психологи
* Коучи
* Массажисты
* Нутрициологи

## Боли клиента (говорить именно так, без жаргона)

* Клиенты приходят один раз и не возвращаются
* Запись идёт через WhatsApp — хаос и потеря заявок
* Нет времени напоминать каждому клиенту вручную
* Не знаю как это автоматизировать, боюсь потратить деньги впустую
* Каждый месяц не знаю, будет ли загрузка

## Что получает клиент (результат, не инструменты)

* Клиенты возвращаются сами — без её участия
* Запись работает 24/7, даже когда она на процедуре
* Нет потерянных заявок и ночных сообщений
* Понятная система без технических сложностей

## Основные функции MVP

* Telegram-бот
* Приветственное меню
* Квиз по подбору процедуры
* Форма заявки на консультацию
* База клиентов
* Автоматические напоминания клиентам
* Уведомления косметологу о заявках

## Текущие файлы

* `index.html` — **главный лендинг-портфолио** (активная разработка): герой, статистика, проблема, решение, процесс, кейсы, тарифы, обо мне, отзывы, FAQ, форма, финальный CTA. Фото Ольги встроено base64.
* `landing-cosmetolog.html` — заморожен, референсный вариант лендинга
* `vizitka-cosmetolog.html` — визитка Ольги как AI-консультанта: фото, специализация, контакты

## Что сделано в лендинге (index.html)
