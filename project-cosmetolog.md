# AI Client Return System — Описание продукта

## Что это

AI-система для косметологов, которая помогает автоматически возвращать клиентов и увеличивать количество повторных записей.

Система работает через Telegram-бота и автоматические сценарии общения с клиентами.

## Для кого

Основная аудитория:
* Частные косметологи
* Косметологические кабинеты
* Небольшие бьюти-студии

В будущем:
* Психологи
* Коучи
* Массажисты
* Нутрициологи

## Какую проблему решает

После первой или второй процедуры многие клиенты перестают записываться повторно.
У косметологов нет времени вручную напоминать о себе каждому клиенту.
Из-за этого теряется большое количество повторных продаж.

## Основные функции MVP

* Telegram-бот
* Приветственное меню
* Квиз по подбору процедуры
* Форма заявки на консультацию
* База клиентов
* Автоматические напоминания клиентам
* Уведомления косметологу о заявках
