# Портфолио сайт — Ольга, AI-консультант

## Что это за проект
Личный портфолио-сайт Ольги — AI-консультанта по автоматизации и росту бизнеса.
Сайт объединяет личную визитку, лендинги услуг и всё будущее портфолио в одном месте.
Размещён бесплатно на GitHub Pages, доступен по публичной ссылке.

## Для кого
- Потенциальные клиенты — малый бизнес и эксперты, которые хотят внедрить AI
- Партнёры и коллеги, которым нужно быстро понять чем занимается Ольга
- Все, кому она отправляет ссылку-визитку

## Стиль дизайна
- **Тёмная тема** — фон #020d0d (почти чёрный с зелёным оттенком)
- **Акцентные цвета** — бирюза `#00e5c3`, зелёный `#00ff88`, синий `#0099ff`
- **Анимированный фон** — плавающие математические и AI-символы на canvas
- **Эффекты** — свечение, glassmorphism-карточки, градиентные заголовки, shimmer-анимация
- **Адаптивная вёрстка** — корректно отображается на телефоне и компьютере
- **Без фреймворков** — чистый HTML + CSS + JavaScript

## Технологии
- HTML / CSS / JavaScript
- Canvas API для анимированного фона
- GitHub Pages — бесплатный хостинг
- Git — версионирование и деплой

## Как отправить изменения на сайт
```
git add .
git commit -m "описание изменений"
git push
```
Сайт обновится через 1-2 минуты.
