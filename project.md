# Мой проект — Портфолио сайт

## Описание
Личный портфолио-сайт Ольги. Набор красивых HTML-страниц с тёмным дизайном в бирюзово-зелёно-синих цветах. Размещён бесплатно на GitHub Pages.

## Страницы

| Страница | Описание | Ссылка |
|----------|----------|--------|
| Главная | Список всех проектов карточками | /my-first-project/ |
| Визитка | Личная цифровая визитная карточка | /vizitka.html |
| Умные калькуляторы | Лендинг для бизнеса | /landing.html |
| AI-консультант | Лендинг по автоматизации и AI | /ai-consultant.html |

## Технологии
- Чистый HTML + CSS + JavaScript (без фреймворков)
- Canvas API для анимированного фона
- GitHub Pages для хостинга
- Git для версионирования

## Как открыть локально
Просто открыть любой .html файл в браузере двойным кликом.

## Как отправить изменения на сайт
```
git add .
git commit -m "описание изменений"
git push
```
Сайт обновится через 1-2 минуты.
