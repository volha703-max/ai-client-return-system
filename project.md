# Портфолио сайт — Ольга, AI-специалист

## Что это за проект
Личный портфолио-сайт Ольги — AI-специалиста по автоматизации.
Сайт объединяет личную визитку, лендинги услуг и всё будущее портфолио в одном месте.
Размещён бесплатно на GitHub Pages, доступен по публичной ссылке.

## Для кого
- Потенциальные клиенты — косметологи и другие эксперты
- Все, кому она отправляет ссылку-визитку

## Страницы — уже есть

| Страница | Описание |

| Визитка | Личная цифровая визитка с фото и анимацией |
| Умные калькуляторы | Лендинг по разработке калькуляторов для бизнеса |
| Визитка косметолога | Персональная визитка для ниши косметологов |
