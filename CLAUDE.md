## Правила работы с проектом

### Технические
- Каждая страница — один самодостаточный HTML-файл (CSS и JS внутри)
- Никаких внешних библиотек, фреймворков и CDN — только чистый HTML/CSS/JS
- `vizitka.html` очень большой (base64 фото) — читать только нужные части с offset/limit
- После любых изменений: git add + commit + push
- После добавления новой страницы — сразу добавить карточку в `index.html`

### Дизайн
- Тёмный фон (#020d0d), светлый текст (#e0fff8)
- Акцентный цвет — бирюзово-голубой (#00e5c3) и его оттенки
- Все новые страницы строго в едином стиле с существующими
- Адаптивная вёрстка обязательна — всё должно хорошо выглядеть на мобильных (от 320px)

### Текст и коммуникация
- Стиль текста на страницах: простой, дружеский, без канцелярита
- Перед созданием нового файла — сначала объяснить Ольге что планируется сделать, дождаться подтверждения
- Объяснять понятно, без лишнего технического жаргона
